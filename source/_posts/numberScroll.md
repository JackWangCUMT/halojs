## 功能

老虎机数字滚动

> author leeenx
> version 1.0.0
> data: 2015-11-11

## Official Usage

```javascript
$("#container").numberScroll(
	{
        round:3,
        duration: 1,
        direction: 'alternate',//
        l2r:1,
        delay: .3,
        groupNum:2,
        diffrentiation:1,
        minLen:0,
        undreg:''
    }
);
```
